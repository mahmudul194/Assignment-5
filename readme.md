1. What is the difference between var, let and const?
Answer : var is the old way to create or declare variables, Can be changed and declared again. let is the New Way to create a variable, Can be changed but not declared again in the same place. const is used for values that can't be changed.

2. What is the spread operator?
Answer : The spread operator is used to copy or combine arrays and objects.

3. What is the difference between map(), filter() and forEach() loop?
Answer: map() changes every item and makes a new array. filter() selects items that match a condition. forEach() loops through items but does not create a new array.

4. What is an arrow function?
Answer : An arrow function is a short way to write a function in javascript.

5. What are templete literals?
Answer : Templete literals are used to insert variables inside a string using backticks.
